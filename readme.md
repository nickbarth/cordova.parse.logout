## Cordova iOS Logout

## About this Plugin

Logout iOS device from Parse.

## Using the Plugin

Example:

```
window.iOSLogout(function () {
  console.log('Logout Current Parse User');
});
```

## Adding the Plugin ##

```
  cordova plugin add https://github.com/nickbarth/cordova.parse.logout
```
